2: Dictionaries

Let's say we have a set of students, and the scores each of them got on a math test:

If we want to store the students and their scores, we could use two lists:

xxxxxxxxxx
 
students = ["Tom", "Jim", "Sue", "Ann"]
scores = [70, 80, 85, 75]

If we wanted to figure out what score Sue got on the test, we'd first have to write a loop to find the index corresponding to Sue in the students list. We'd then have to find the value associated with that index in scores. Here's the code:

xxxxxxxxxx
 
indices = [0,1,2,3]
name = "Sue"
score = 0
for i in indices:
    if students[i] == name:
        score = scores[i]
print(score)

This is a complex piece of code for something simple -- we just want to find the value associated with a name.

To accomplish this in an easier way, we can use a dictionary. A dictionary is like a list in that it has indexes, but each index can be any value, including a string.

We make a dictionary with curly braces ({) instead of using square brackets ([) like lists. We can make an empty dictionary like this:

xxxxxxxxxx
 
scores = {}

Once we create a dictionary, we can add values by specifying the index we want to add the value for:

xxxxxxxxxx
 
scores["Tom"] = 70

The code above will add the value 70 to scores for the index Tom. If we want to get the test score for Tom, we would just use:

xxxxxxxxxx
 
scores["Tom"]

The code above would return 70. This is because we associated the value 70 with the index Tom in the dictionary scores. We use square brackets ([) to add values to dictionaries or look up values.

We can add the rest of the student scores in the same way:

xxxxxxxxxx
 
scores["Jim"] = 80
scores["Sue"] = 85
scores["Ann"] = 75

3: Practice populating a dictionary

To create a dictionary, you first define it with curly braces ({), then you add values for specific indexes. The indexes are also called dictionary keys:

xxxxxxxxxx
 
students = {}
students["Jerry"] = 60

In the example above, we create an empty dictionary called students, then specify that the dictionary key Jerry is associated with the value 60.

If we wanted to find the value associated with the dictionary key Jerry, we'd lookup Jerry in the dictionary students:

xxxxxxxxxx
 
print(students["Jerry"])

The code above would print out 60.

A dictionary key can be a string, integer, or float:

xxxxxxxxxx
 
students[10] = 100

5: Defining a dictionary with values

So far, we've been creating a dictionary and adding values in multiple steps:

xxxxxxxxxx
 
students = {}
students["Tom"] = 60
students["Jim"] = 70

This is cumbersome when we want to add multiple dictionary keys. We can instead combine creation and adding values into one step:

xxxxxxxxxx
 
students = {
    "Tom": 60,
    "Jim": 70
}

In the above code, we create a dictionary, but define that the key Tom is associated with the value 60, and the key Jim is associated with the value 70.

We do this by specifying the dictionary key, then a colon (:), then the value. We separate each key/value pair with a comma. We could add more students if we wanted to:

xxxxxxxxxx
 
students = {
    "Tom": 60,
    "Jim": 70,
    "Sue": 85,
    "Ann": 80
}

You can specify as many key/value pairs as you want this way.

6: Modifying dictionary values

Just like with lists, we can modify the values in a dictionary:

xxxxxxxxxx
 
students = {
    "Tom": 60,
    "Jim": 70
}

We can replace the value associated with a key:

xxxxxxxxxx
 
students["Tom"] = 65

The code above would change the value associated with the key Tom to 65.

We can also modify an existing value:

xxxxxxxxxx
 
students["Tom"] = students["Tom"] + 5

The code above would add 5 to the value associated with the key Tom.

8: The else statement

We learned about the if statement in a previous mission. With the if statement, we run a segment of code if a condition is True:

xxxxxxxxxx
 
if temperature > 50:
    print("It's hot!")

In the code above, the if statement checks to see if the variable temperature is greater than 50, and prints out It's hot! if it is.

Let's say we also want to print out something if the temperature isn't greater than 50. We can check for this by seeing if the temperate is less than or equal to 50:

xxxxxxxxxx
 
if temperature > 50:
    print("It's hot!")
if temperature <= 50:
    print("It's cold!")

If temperature is greater than 50, we'll see It's hot!, and if it's less than or equal to 50, we'll see It's cold!. Another way to say this is that we print one statement when temperature > 50 equals True, and another statement when temperature > 50 equals False.

This case comes up often in programming, where we want to do something if a condition is True, and something else when a condition is False.

A simpler way to express what we want is to use the else statement:

xxxxxxxxxx
 
if temperature > 50:
    print("It's hot!")
else:
    print("It's cold!")

The code above is much simpler than before, but it results in the same outcome. If temperature > 50 is True, then it executes the code in the if statement block. If temperature > 50 is False, then it executes the code in the else statement block.

It's not possible to execute both the if statement block and the else statement block. So the code above will only print It's hot! or It's cold!, never both.

9: Practicing with the else statement

Else statements allow us to simplify our code. Here's an example:

xxxxxxxxxx
 
scores = [80, 100, 60, 30]
high_scores = []
low_scores = []
for score in scores:
    if score > 70:
        high_scores.append(score)
    else:
        low_scores.append(score)

The above code will add a score to high_scores if it's greater than 70, and to low_scores otherwise.
